# OTOPOS - Aplikasi Kasir Bengkel Android

## Cara Install
1.  Download file `OTOPOS.apk`
2.  Buka file di HP → Klik `Install`
3.  Kalau ada peringatan "Sumber tidak dikenal" klik `Izinkan`
4.  Selesai. Buka aplikasi OTOPOS di HP

## Fitur
1.  Login Multi User
2.  Dashboard Ringkasan Omset
3.  Kasir - Kendaraan Ditangani
4.  Daftar Part + Stok
5.  Input Part
6.  Daftar Mekanik
7.  Laporan + Grafik
8.  Setting + Bantuan WA
