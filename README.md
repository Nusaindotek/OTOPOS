# OTOPOS - Aplikasi Kasir Bengkel Android

## Cara Install
1.  Download file `OTOPOS.apk`
2.  Buka file di HP → Klik `Install`
3.  Kalau ada peringatan "Sumber tidak dikenal" klik `Izinkan`
4.  Selesai. Buka aplikasi OTOPOS di HP kamu

## Fitur
1.  Login Multi User
2.  Dashboard Ringkasan Omset & Service
3.  Kasir - Kelola Kendaraan Ditangani
4.  Daftar Part + Stok Otomatis
5.  Input/Edit Part
6.  Daftar Mekanik + Status Kerja
7.  Laporan + Grafik Omset 7 Hari
8.  Setting Data Bengkel
9.  Bantuan WA & Email
